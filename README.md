# 🎮 Jogo da Velha (Tic-Tac-Toe)

Projeto desenvolvido em Python com foco em organização de código, modularização e boas práticas de desenvolvimento.

---

## 📌 Sobre o projeto

Este é um jogo da velha executado no terminal, onde dois jogadores se alternam para realizar jogadas até que haja um vencedor ou empate.

O projeto foi refatorado com o objetivo de melhorar a estrutura do código, separando responsabilidades em diferentes módulos.

---

## 🚀 Funcionalidades

* ✅ Jogo para dois jogadores (X e O)
* ✅ Validação de jogadas
* ✅ Verificação automática de vitória
* ✅ Detecção de empate
* ✅ Interface simples no terminal
* ✅ Código organizado em módulos
